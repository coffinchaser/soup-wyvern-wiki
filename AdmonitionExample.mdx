# Example MDX with Admonitions

## Different Types of Admonitions

<Admonition type="note" title="Important Note">
  This is a note admonition with some additional information.
</Admonition>

<Admonition type="tip" title="Pro Tip">
  Here's a helpful tip for improving your workflow.
</Admonition>

<Admonition type="warning" title="Be Careful">
  This is a warning about potential issues you might encounter.
</Admonition>

<Admonition type="danger" title="Critical Warning">
  This indicates a serious problem that needs immediate attention.
</Admonition>
